                            function reverseString(userString) {
                            
                                let revString = [];
                            
                                // Reverse string using a for loop
                                for(let index = userString.length - 1; index >= 0; index --) {
                                    revString += userString[index];
                                }
                            
                                return revString;
                            
                            }

                                // Show the alert box

The Code is structured in three functions.

getValue( )

Our first main function, getValue( ), is our controller function responsible for retrieving the string the user input on the webpage.

First, this function adds the class of “invisible” to the HTML elements which are responsible for showing the alert box to the user, both for a successful string reversal and as a warning to inform the user their string length is too short.

Second, it retrieves the string the user entered and checks to see if its length exceeds one character. If is does, then our second and third functions, located at lines 31 and 47, are called to display a reversed string to the user.
If the string entered by the user is only one character in length, an alert is displayed to the user, by removing the “invisible” class on the HTML element which represents the warning alert box.

reverseString( )

Our second function, reverseString(userString), is our logic function and takes a single parameter which is the string entered by the user.

It loops over the length of the string entered, takes the index location of each character in the string, adds it to an empty array called revString, and finally returns the new array.

displayString( )

Our third function, displayString(revString), is our view function and is responsible for displaying the reversed version of the string entered, back to the user.

It also takes a single parameter, which is the reversed string result from our second function, and writes a message to the page which shows the newly reversed string.

Finally, it removes the “invisible” class from the HTML element which represents the successfully reversal alert box, so that this is displayed to the user.
